# Дизайн-документ: Эпохи и апгрейды

Этот файл оставлен как совместимая входная точка для старых ссылок.

Актуальная структура разделена на три документа:

* [eras_overview.md](eras_overview.md) -- дизайн-описание эр, fantasy,
  материалы, транспорт, дороги, торговля и смена масштаба.
* [era_progression_gates.md](era_progression_gates.md) -- точные условия
  переходов между эрами.
* [building_progression.md](building_progression.md) -- прогрессия зданий,
  уровни, замены и поперечные цепочки через несколько эр.

Детальный сценарий Палаточной эры остается в
[tent_era_survival.md](tent_era_survival.md).
